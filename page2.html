CSS

CSS, or cascading style sheets, is (or are?) used to add some styling finesse to basic HTML pages. That first page was pretty dire, and probably triggering if you were alive in the 90s when most sites looked like that, and this page should hopefully look a little better. There are people who really excel in making pages look amazing with CSS, though I'm not one of them yet. By this stage of the course (week 1-2) I'd just grasped the basics of HTML and was starting to learn CSS, so I only understood very elementary controls for styling, such as fonts and background colours. In the upcoming pages I'll start to use flexbox to improve the layout.
